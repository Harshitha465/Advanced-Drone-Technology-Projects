# Advanced Drone Technology Projects

This repository contains projects completed during the Winter Training Program on Space Science Technology.

## Project Report

The complete report is available in:

- DRONE TECHNOLOGY REPORT.pdf

## Projects Included

### Airfoil Exploration and Performance Analysis
- Lift and drag analysis
- Angle of attack study
- Airfoil comparison using NASA Airfoil Simulator

### Drone Telemetry Navigation System
- YOLOv8 object detection
- Target lock functionality
- Real-time monitoring concepts

### Computer Vision Techniques for Drone Applications
- Landing Pad Finder
- Horizon Leveler
- Obstacle Alert System
- Night Vision Booster
- Motion Blur Simulator

### VTOL Aircraft Control System Design
- Pixhawk Flight Controller
- ArduPilot
- Sensor Integration
- Flight Mode Transition Analysis
- Stability and Fail-safe Systems

## Technologies Used

- Python
- OpenCV
- NumPy
- YOLOv8
- ArduPilot
- Pixhawk
- NASA Airfoil Simulator
